## Overview

BlockBait is a cutting-edge Chrome extension designed to protect users from phishing attacks. By leveraging advanced machine learning techniques and intuitive web technologies, it identifies and blocks phishing websites in real-time. The extension provides an additional layer of security for users by analyzing browsing history, cross-referencing whitelists, and scanning HTML for malicious intent.


## Key Features

- **Phishing Detection**: Utilizes machine learning algorithms, including support vector machines (SVM), to detect phishing attempts based on website patterns

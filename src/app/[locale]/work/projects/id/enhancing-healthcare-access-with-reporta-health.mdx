## Overview

Reporta Health is a web and mobile application designed to improve healthcare service delivery in Nigeria. By leveraging technology, the platform empowers health workers and users to access information about registered and unregistered health facilities, report irregularities, and enhance overall service quality. The solution emphasizes user-friendly navigation, robust data reporting, and advanced map integrations to create a seamless experience for users.

## Key Features

### Mobile Application
- **Facility Search**: Enables users to search for health facilities, view available services, and access details like operating hours and contact information.
- **Real-Time Reporting**: Provides a platform for users to report unregistered or non-compliant facilities, aiding regulatory bodies in ensuring service quality.
- **Navigation**: Integrated with Leaflet and Google Maps, allowing health workers to navigate directly to facilities.
- **Statistics Dashboard**: Displays detailed statistics on health facility distributions, types of services, and reported cases for data-driven decision-making.

### Web Application
- **Admin Dashboard**: For regulatory bodies to review reports, manage data, and generate insights on healthcare service delivery trends.
- **User Management**: Supports multi-level roles, including administrators and field health workers, with tailored interfaces for each.
- **Data Visualization**: Interactive charts and maps to track facility status, service availability, and regional disparities.

## Technologies Used

- **React Native**: For building the cross-platform mobile application.
- **Next.js**: To power the web application with server-rendered pages for improved SEO and performance.
- **Custom CSS**: For creating a visually appealing and responsive UI.
- **Zustand and React Query**: Ensuring efficient state and server data management.
- **Leaflet and Google Maps**: Integrating map functionalities for searching and navigating health facilities.

## Challenges and Learnings

Implementing a map-based search and reporting system presented unique challenges, such as ensuring accurate geolocation data and handling offline scenarios for remote health workers. Another hurdle was optimizing the system for large datasets, including thousands of facility records, without compromising performance. Through iterative development and real-world testing, the application now provides a reliable and scalable platform.

## Outcome

Reporta Health has significantly improved healthcare access and quality by empowering users to contribute to the healthcare ecosystem. Health workers can navigate, search, and report facilities efficiently, while regulators benefit from actionable insights through the web application. This project underscores the transformative power of technology in addressing systemic challenges in healthcare service delivery.
